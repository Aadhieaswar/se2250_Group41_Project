# Gunslinger's Journey - Unity Game

### Requirements
- Unity `2019.4.17`
- Visual Studio 2019 - text editor (Recommended)

### Description
- The game is present in the `New Unity Project` folder
- Play from the `MainMenu` scene in the `_Scenes` folder to start the game
